# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in CineStream, please report it responsibly:

- **Email:** Create an issue on GitHub with the `security` label
- **Do not** publicly disclose security issues before they are fixed

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.0.x   | ✅ Yes    |
| 1.0.x   | ❌ No     |

## APK Integrity

All official CineStream APK files are signed with our release key. 
Download only from the [official GitHub releases](https://github.com/D13056/CineStream/releases).
